# FutureTrading

MarketSession capture, targets, grading, and metrics.

Topics:
- MarketOpenCaptureService: 11 futures + TOTAL, `session_number` grouping
- Entry/targets: `compute_targets_for_symbol`
- Backtest stats: fields and update points
- Intraday metrics: high/low %, close/range
- VWAP minutes: session + rolling windows

See `DATA_CONTRACTS.md` for payload schemas and `DB_SCHEMA.md` for model fields.
